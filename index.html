What's inside

Everything you need. Nothing you don't.

Built for power users who care about security — without sacrificing usability.

Security
🔐

KDBX 4.1 Encryption

AES-256 or ChaCha20 ciphers with Argon2 key derivation — the gold standard KDBX vault format.

🧬

Biometric Unlock

Unlock your vault instantly with Face ID, Touch ID, or fingerprint — no master password typing required.

🔑

TOTP / 2FA Built-in

Generate one-time passwords with live countdown timers — no separate authenticator app needed.

🛡️

HIBP Breach Check

Check passwords against Have I Been Pwned using k-anonymity — only 5 SHA-1 chars ever leave the device.

🧹

Secure Clipboard

Auto-clears clipboard after a configurable timeout and wipes on vault lock.

Vault
📂

Groups, Tags & Search

Organize with a full group tree, tags, favorites, and instant full-text search across all fields.

📋

Entry Templates

Login, Credit Card, SSH Key, Server, Database, API Key, TLS Cert, and more — ready to use out of the box.

📎

Encrypted Attachments

Attach files to any entry with in-app PDF, image, and text preview — all encrypted inside the vault.

🔄

Flexible Import

Import from 1Password, Bitwarden, LastPass, or any generic CSV / JSON export in seconds.

♻️

Recycle Bin

Deleted entries go to a recycle bin first — restore anything accidentally removed, entry by entry.

Sync & Backup
☁️

Opt-in Cloud Sync

Sync via Google Drive, Dropbox, OneDrive, iCloud, Nextcloud, or WebDAV — your choice, never forced.

🔀

3-Way KDBX Merge

Intelligent merge resolves conflicts across devices; a dialog lets you pick the winning change.

💾

Auto-Backup

Scheduled backups with versioned filenames and a configurable retention policy keep your vault safe.

📶

Offline Queue

Failed syncs are queued and automatically retried the moment connectivity is restored.

Platform
🖥️

macOS Desktop

System tray icon, global hotkey (⌘⇧B), and DMG installer — first-class desktop experience.

🌐

Browser Extension Bridge

Chrome MV3 & Firefox MV2 extension autofills credentials directly from the native app — no separate vault.

🤖

Android Autofill Service

Fills usernames and passwords in any Android app via the Autofill Service API — no clipboard required.

🍎

iOS AutoFill Extension

QuickType bar integration suggests the right credential the moment iOS detects a login field.
